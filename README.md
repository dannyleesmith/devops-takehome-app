# Devops Takehome app

Env variables:
 - DATABASE_NAME
 - DATABASE_HOST
 - DATABASE_PORT
 - DATABASE_USERNAME
 - DATABASE_PASSWORD
 
 App runs on port: 3000
 MySQL table: ping_pong
 
 To start the app: `node index.js`
 
 ## Endpoints
 ### `/ping`
 Should return `{"ping": "pong"}`
 
 ### `/pong`
 Should return `{"pong": "ping"}`
